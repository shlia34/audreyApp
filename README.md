# audrey-app

「オードリーのオールナイトニッポン」のHPの過去放送のブログから、日付・タイトル・画像・HPのブログへのリンク をスクレイピングしました

## 機能

・スクレイピングしたものを表示、ページネーション機能

・お気に入り機能・一覧表示

・インクリメンタルサーチ

・Userのログイン、ログアウト、サインアップ（devise）

・Userのアイコン画像、プロフィール欄を追加

・放送に対してのコメント機能
